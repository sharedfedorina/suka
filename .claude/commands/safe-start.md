# Безпечний старт роботи

Виконай наступні кроки:

1. **Прочитай CLAUDE.md ПОВНІСТЮ**
   ```
   cat .claude/CLAUDE.md
   ```
   Переконайся що розумієш:
   - Структуру проекту
   - Критичні правила
   - Маппінг плейсхолдерів
   - Модульну архітектуру

2. **Перевір стан git**
   ```bash
   git status
   ```
   Якщо є uncommitted зміни → спитай людину що робити!

3. **Перевір чи працює сервер**
   ```bash
   node server.js
   ```
   Має запуститися на порту 3000

4. **Запитай людину: "Що робимо далі?"**
   
   НЕ починай кодити без підтвердження!

5. **Після отримання завдання:**
   - Розбий на маленькі кроки (1-2 файли за раз)
   - Commit після КОЖНОГО кроку
   - Повідомляй про прогрес

---

**КРИТИЧНО: НЕ ПОЧИНАЙ РОБОТУ БЕЗ ЦІЄЇ КОМАНДИ!**
