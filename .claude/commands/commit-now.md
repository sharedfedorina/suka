# Негайний commit

Виконай:

```bash
# Подивись що змінилося
git status

# Додай всі зміни
git add .

# Створи commit з описовим повідомленням
git commit -m "Progress: [опиши що зробив коротко]"
```

**Приклади good commit messages:**
- `feat: додав hero image upload`
- `fix: виправив маппінг heroTitle`
- `refactor: розбив tabs.js на модулі`
- `style: форматування sections/hero.html`

**Після commit:**
- Повідом людину що зроблено
- Запитай чи продовжувати
